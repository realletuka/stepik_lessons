-- Тестовый сценарий 1.1 Применение валидного кода купона на скидку
Проверка сценария применения кода на скидку, когда пользователь ввел верный код
Предусловие
Пользователь залогинен
Товар http://selenium1py.pythonanywhere.com/ru/catalogue/the-shellcoders-handbook_209/ добавлен в корзину из каталога
У пользователя есть действующий код купона на скидку (ссылка на условия применения кодов или программу лояльности)
Шаги:
Шаг 1 Перейти в корзину, нажав кнопку «Посмотреть корзину» в правом верхнем углу/ ОР Открыта страница корзины с добавленными товарами
Шаг 2 На странице Корзины нажать кнопку «У меня есть код скидки»/ ОР Открывается поле для ввода кода
Шаг 3 В поле ввести корректный (действующий) код купона и нажать «Применить»/ ОР Код для скидки применился и сумма в корзине изменилась с учетом скидки по данному коду

-Пользовательский сценарий 2 Изменение пользователем пароля аккаунта
-—Тестовый сценарий 1.1 Проверка возможности изменение пароля аккаунта на странице «Профиль» с валидными данными паролей
Предусловия:
Пользователь залогинен
Шаги:
Шаг 1 Войти в раздел Аккаунт, нажав кнопку «Аккаунт» в правом верхнем углу/ ОР Открылась страница «Аккаунт» и раздел «Профиль»
Шаг 2 Под таблицей с данными аккаунта нажать синюю кнопку «Изменить пароль»/ ОР Открылась страница «Изменить пароль»
Шаг 3 В поле «Старый пароль» ввести валидный старый пароль/ ОР Поле «Старый пароль» заполнено
Шаг 4 В поле «Новый пароль» ввести новый пароль (согласно требованиям)/ОР Поле «Новый пароль» заполнено
Шаг 5 В поле «Подтверждение нового пароля» повторно ввести новый пароль и нажать кнопку «Сохранить»/ ОР Осуществлен переход на страницу «Профиль», отобразилось сообщение «Пароль обновлен»

-Пользовательский сценарий 2 Изменение пользователем пароля аккаунта
-—Тестовый сценарий 1.1 Проверка возможности изменение пароля аккаунта на странице «Профиль», где значения полей «Новый пароль» и «Подтверждение пароля» не совпадают.
Предусловия:
Пользователь залогинен
Шаги:
Шаг 1 Войти в раздел Аккаунт, нажав кнопку «Аккаунт» в правом верхнем углу/ ОР Открылась страница «Аккаунт» и раздел «Профиль»
Шаг 2 Под таблицей с данными аккаунта нажать синюю кнопку «Изменить пароль»/ ОР Открылась страница «Изменить пароль»
Шаг 3 В поле «Старый пароль» ввести валидный старый пароль/ ОР Поле «Старый пароль» заполнено
Шаг 4 В поле «Новый пароль» ввести значение нового пароля (согласно требованиям)/ОР Поле «Новый пароль» заполнено
Шаг 5 В поле «Подтверждение нового пароля» ввести значения, не совпадающие со значениями, указанными в шаге 4 и нажать кнопку «Сохранить»/ ОР Подсвечено красным цветом поле «Подтверждение нового пароля», появилось сообщение об ошибке «Два поля с паролями не совпадают», Пароль не обновился.
